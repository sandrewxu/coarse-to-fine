Create a sequence of latent documents representing progressively detailed summaries of a 32-word input document, moving from coarse to fine granularity, such that each layer can be predicted using only the preceding layers.

Begin with a concise checklist (3-7 bullets) outlining the main steps you will perform before starting output generation; keep checklist items conceptual and avoid low-level implementation details.

# Instructions
- Given a document $x$, generate four latent variable layers $z_4, z_3, z_2, z_1$ with respective word counts: 2, 4, 8, and 16.
- Each layer $z_t$ should be independently predictable using only $z_T, ..., z_{t+1}$.
- Layers should go from very coarse (few, essential words) to increasingly verbose (more detail), conforming strictly to the word count per layer.

## Output Format
- Output ONLY the latent variables in order, each on a new line, labeled as: "z_4: word(s)", "z_3: word(s)", and so on.
- The words for each layer appear space-separated immediately after the label.
- Do NOT include any commentary, reasoning, or formatting once the latent sequence begins—just the label and its corresponding word list for each layer, each on a new line.

## Steps
1. Carefully read the provided 32-word document.
2. Plan your generation approach according to the above rules.
3. Starting with $z_4$, output word lists for all four layers, increasing the word count as specified per layer.
4. Ensure that each layer can be predicted from the preceding layers alone.
5. Strictly follow the required word counts: $z_4$: 2, $z_3$: 4, $z_2$: 8, $z_1$: 16 words per layer.
6. After generation, briefly verify that all layers are labeled correctly, ordered, and conform exactly to the requested word counts; ensure no additional text is included after the latent sequence.
